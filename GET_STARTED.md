# How to use Tiimber

The project of this file is for purpose to create a little blog.

## Requierement

For use Tiimber, you need intall PHP 7 and composer

## Intalling Tiimber

`in composer.json`

```json
{
  "require": {
    "ndufreche/tiimber": "dev-master"
  },
  "autoload": {
    "psr-4": {
      "Blog\\": "Blog/"
    }
  }
}
```
